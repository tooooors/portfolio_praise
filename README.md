# ほめほめ社内SNS
「褒める」を見える化したオリジナル社内SNSです。<br>
褒めるごとにポイントが加算されるシステムです。<br>
社員同士で褒め合うことによって社員が自信をつけ、社内の雰囲気を向上させることができるのではないかと考え作成しました。<br>
褒められていない人を褒めるほど多くのポイントが加算される仕組みにすることで、褒められる人に偏りがでないようにしました。<br>
